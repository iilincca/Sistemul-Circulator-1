Capitolul VI – Prezentarea proiectului

✔️ VI.1 – Impresia generală a proiectului

„Sistemul Circulator” este o aplicație educațională dezvoltată cu scopul de a explica simplu și vizual cum funcționează sistemul circulator uman. Interfața este aerisită și clar structurată, fiind ușor de parcurs chiar și de către utilizatori mai puțin familiarizați cu aplicațiile web. Utilizatorul poate învăța prin text, imagini, animații video și quiz – adică prin toate tipurile de conținut esențiale unui software educațional modern. Funcționalitățile sunt bine delimitate, iar aplicația este complet portabilă.

✔️ VI.2 – Documentația proiectului

  • ✔️ Informații generale: Aplicația este dezvoltată în HTML, CSS și JavaScript – cele mai uzuale și portabile tehnologii pentru aplicații web statice. Este destinată utilizării offline, în special în mediul educațional.
  • ✔️ Ghid de instalare și utilizare:
    • Toate fișierele (HTML, CSS, JS, imagini, videoclipuri) se păstrează într-un singur folder.
    • Se deschide fișierul Sistemul Circulator.html într-un browser modern (Chrome, Firefox, Edge).
    • Aplicația rulează fără conexiune la internet, atâta timp cât fișierele video sunt locale.
  • ✔️ Arhitectura aplicației:
    • Sistemul Circulator.html – fișierul principal, care definește structura conținutului și încarcă resursele externe.
    • style.css – fișierul de stilizare: definește aspectul vizual, culorile, fonturile, layout-ul și adaptarea la diferite dimensiuni de ecran (responsive).
    • script.js – fișierul de JavaScript care conține logica funcțională: quiz-ul, comutarea limbii, validarea și afișarea scorului.
    • Imaginile și videoclipurile sunt stocate local și sunt accesibile fără conexiune la internet.
  • ✔️ Justificarea tehnologiilor alese:

    Tehnologiile HTML, CSS și JavaScript au fost alese datorită:

    • ✔️ Portabilității – funcționează în orice browser, pe orice dispozitiv, fără instalare.
    • ✔️ Separării clare a logicii de prezentare – codul e împărțit în fișiere separate (structură, stil, funcționalitate).
    • ✔️ Ușurinței de întreținere – fișierele pot fi modificate fără afectarea funcționalității globale.
    • ✔️ Accesibilității pentru învățare – elevii și profesorii pot înțelege ușor cum este construit proiectul.
